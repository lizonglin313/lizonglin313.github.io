MarkDown数学公式

2019-01-22 3 min read # 建站 # 杂项

我们在使用markdown编辑时，是会有数学公式语法的。

基本语法对照表

  • 嵌入行内的公式用$包裹，如：行内公式展示a=b+ca = b + ca=b+c，效果为：

    行内公式展示a=b+ca=b+c

  • 换行居中独立模块展示可以用$$包裹，如：模块展示$$a = b + c$$，效果为：

    模块展示

    a=b+ca=b+c

    ​
